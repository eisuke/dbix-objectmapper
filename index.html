dbix-objectmapper by eisuke

An implementation of the Data Mapper pattern (object-relational mapper).

DBIx::ObjectMapper is a implementation of the Data Mapper pattern. And abstraction layer for database access. Concepts and interfaces of this module borrowed from SQLAlchemy. http://www.sqlalchemy.org/
